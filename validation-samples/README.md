# Validation Samples

These audio files and transcripts were used to verify that the forced-alignment implementation produces correct word-level timestamps.

Each sample consists of:
- `.mp3` — the audio recording
- `.md` — the known transcript text
- `.truth.json` — word-level timestamps independently verified as correct
(generated by a separate alignment tool and spot-checked by ear)

The truth files are not output from this CLI. They are reference data:
the implementation is considered correct when its output matches these
timestamps within acceptable tolerance.
